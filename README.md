# CorBas (Desktop Version)

CorBas is a **desktop corpus analysis application** developed by **Muhammad Hussein Mahdi Mohsen** at the **University of Basra**.

The software is designed to assist researchers, students, and linguists in analyzing textual corpora through a variety of computational and linguistic tools, including frequency analysis, concordance, and graphical visualizations.

## Features
- Word frequency analysis
- Concordance (KWIC)
- Graphical analysis (e.g., lexical dispersion plots)
- Corpus upload and processing
- Exportable results (e.g., highlighted PDFs, Excel sheets, Word documents)
- And many other features

## Installation
Download the latest release from:
https://github.com/mhdy54370-hub/CorBas-updates/releases

## Usage
1. Launch the application
2. Upload your corpus
3. Select an analysis tool
4. View and export the results
